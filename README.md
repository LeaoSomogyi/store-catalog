# StoreCatalog

Esse projeto tem como intuito servir como estudos para a matéria do MBA de Arquitetura de Integração e Microsserviços

## Getting Started

```
git clone https://github.com/quyk/store-catalog.git
```

### Pré Requisitos

DotNet Core 2.2 -> https://dotnet.microsoft.com/download/dotnet-core/2.2

### Instalando

Na pasta do projeto, rode o comando abaixo para restaurar os pacotes:

```
dotnet restore
```

Agora, rode o comando abaixo para buildar o projeto:

```
dotnet build
```

## Rodando os testes

```
dotnet test
```

## Construído com

* [dotnet core](https://docs.microsoft.com/pt-br/dotnet/core/) - Web API e Class Library
